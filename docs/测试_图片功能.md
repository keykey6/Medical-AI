# 医疗AI智能客服 - 图片上传功能测试指南

## 一、功能概述

本系统新增图片上传与分析功能，支持用户上传图片并获得纯中文的客观描述。

## 二、新增接口说明

### 2.1 图片上传接口

**接口地址**: `POST /api/chat/send_image`

**请求方式**: `multipart/form-data`

**参数说明**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| session_id | string | 否 | 会话ID，不传则自动生成新会话 |
| message | string | 否 | 用户补充提问文字 |
| file | file | 是 | 图片文件（支持JPG/PNG/GIF/BMP/WebP，最大10MB） |

**返回示例**:

```json
{
    "session_id": "test_session",
    "response": "图片中可见皮肤局部有红色斑块状表现。以上描述为客观观察，不构成任何医疗诊断或治疗建议。如您有健康疑虑，请及时前往正规医院就诊。",
    "is_transfer": false,
    "question_type": "医疗咨询",
    "emotion_type": "平静"
}
```

### 2.2 Base64图片上传接口

**接口地址**: `POST /api/chat/send_image_base64`

**请求方式**: `application/json`

**参数说明**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| session_id | string | 否 | 会话ID |
| message | string | 否 | 用户补充提问 |
| image_base64 | string | 是 | Base64编码的图片数据 |

**返回示例**: 同上

## 三、测试步骤

### 3.1 环境准备

1. 确保服务已启动：`python -m uvicorn main:app --host 0.0.0.0 --port 8000`
2. 确保Ollama已启动，且已下载llava:7b模型
3. 检查模型：`ollama list` 应显示 llava:7b

### 3.2 测试命令

#### 测试1：基础图片分析
```bash
curl -X POST http://localhost:8000/api/chat/send_image \
  -F "file=@test_image.jpg" \
  -F "session_id=test_session"
```

#### 测试2：图片+文字提问
```bash
curl -X POST http://localhost:8000/api/chat/send_image \
  -F "file=@test_image.jpg" \
  -F "session_id=test_session" \
  -F "message=帮我看看这个皮肤情况"
```

#### 测试3：测试图片格式校验
```bash
# 使用不支持的格式（如txt文件）
curl -X POST http://localhost:8000/api/chat/send_image \
  -F "file=@test.txt"
# 预期返回：400错误，提示不支持的图片格式
```

#### 测试4：测试图片大小限制
```bash
# 上传大于10MB的图片
curl -X POST http://localhost:8000/api/chat/send_image \
  -F "file=@large_image.jpg"
# 预期返回：400错误，提示图片大小超过限制
```

## 四、合规保障

1. **输出语言**: 所有回复均为中文，无任何英文内容
2. **客观描述**: 仅做事实性描述，不进行疾病诊断
3. **免责声明**: 自动追加中文免责声明
4. **隐私保护**: 图片仅临时处理，不做永久存储

## 五、测试验证标准

| 测试项 | 验证标准 |
|--------|----------|
| 图片上传 | 成功接收图片并返回描述 |
| 中文输出 | 所有回复均为中文 |
| 合规描述 | 无诊断内容，仅客观描述 |
| 免责声明 | 回复末尾有中文免责声明 |
| 格式校验 | 拒绝不支持的文件格式 |
| 大小限制 | 拒绝超过10MB的图片 |
| 会话记录 | 图片对话正确存入数据库 |

## 六、注意事项

1. 首次使用llava模型可能需要较长时间加载
2. 图片分析耗时取决于图片大小和模型性能
3. 建议使用分辨率适中的图片以获得最佳分析效果

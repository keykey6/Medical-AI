## 1. 方案概述

```

│  ┌───────────────────────────────────────────────────────┐  │

│  │  ┌─────────────────────────────────────────────┐   │  │

│                          ▼                                   │

│  │         内网穿透工具（可选，如 cpolar/ngrok）           │  │

│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                          │
              ┌───────────┼───────────┐
              ▼           ▼           ▼
        ┌─────────┐ ┌─────────┐ ┌─────────┐
        │ 手机浏览器 │ │ PWA图标  │ │ 原生App  │
        │ 直接访问  │ │ 主屏幕   │ │ WebView │
        └─────────┘ └─────────┘ └─────────┘
```

### 三种使用方式

| 方式 | 难度 | 体验 | 适用场景 |
|------|------|------|----------|
| **A. 浏览器直接访问** | ⭐ | 良好 | 快速验证，无需安装 |
| **B. PWA 添加到主屏幕** | ⭐⭐ | 接近原生 | 日常使用，有桌面图标 |
| **C. 原生 App 壳** | ⭐⭐⭐ | 原生体验 | 应用商店分发，推送通知 |

> 💡 **推荐路径**：先做 A（5 分钟验证），再做 B（10 分钟优化），需要时再考虑 C。

## 2. 后端保持不变

### 2.1 确认后端运行状态

在你的电脑/服务器上执行：

```bash
# 进入项目目录
cd 医疗知识库/backend

# 启动后端（你现有的启动方式）
python start.py
# 或
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

确认以下地址可访问：

| 地址 | 用途 | 验证方式 |
|------|------|----------|
| `http://localhost:8000/health` | 健康检查 | 浏览器打开 |
| `http://localhost:8000/static/index.html` | 主页面 | 浏览器打开 |
| `http://localhost:8000/docs` | API 文档 | 浏览器打开 |

### 2.2 关键配置确认

打开 `backend/config.py` 或 `.env`，确认：

```ini
# 必须允许外部访问
HOST=0.0.0.0          # 不是 127.0.0.1！
PORT=8000

# CORS 配置（允许手机访问）
# 在 backend/main.py 中确认有：
# app.add_middleware(CORSMiddleware, allow_origins=["*"])
```

如果 `HOST=127.0.0.1`，手机无法连接，必须改为 `0.0.0.0`。

### 2.3 获取后端 IP 地址

```bash
# Windows
ipconfig | findstr IPv4

# Mac/Linux
ifconfig | grep inet | grep -v 127.0.0.1
# 或
hostname -I
```

记录你的内网 IP，例如：`192.168.1.100`

手机访问地址将是：`http://192.168.1.100:8000/static/index.html`

### 3.3 添加移动端 JS 交互

在 `frontend/js/common.js` 末尾追加：

```javascript
// ==========================================
// 移动端交互 — 医疗AI智能客服
// ==========================================

(function() {
    const isMobile = window.innerWidth <= 768 || /Android|iPhone|iPad/i.test(navigator.userAgent);

    if (!isMobile) return;

    // 1. 创建汉堡菜单按钮
    function createMobileMenuBtn() {
        const topbar = document.querySelector('.topbar');
        if (!topbar || topbar.querySelector('.mobile-menu-btn')) return;

        const btn = document.createElement('button');
        btn.className = 'mobile-menu-btn';
        btn.innerHTML = '<i class="fas fa-bars"></i>';
        btn.style.cssText = `
            display: flex;
            align-items: center;
            justify-content: center;
            width: 40px;
            height: 40px;
            background: none;
            border: none;
            color: var(--text-primary);
            font-size: 20px;
            cursor: pointer;
            margin-right: 8px;
        `;

        btn.addEventListener('click', toggleSidebar);
        topbar.insertBefore(btn, topbar.firstChild);
    }

    // 2. 创建遮罩层
    function createOverlay() {
        if (document.querySelector('.sidebar-overlay')) return;

        const overlay = document.createElement('div');
        overlay.className = 'sidebar-overlay';
        overlay.addEventListener('click', closeSidebar);
        document.body.appendChild(overlay);
    }

    // 3. 切换侧边栏
    function toggleSidebar() {
        const sidebar = document.querySelector('.session-sidebar');
        const overlay = document.querySelector('.sidebar-overlay');

        if (sidebar.classList.contains('open')) {
            closeSidebar();
        } else {
            sidebar.classList.add('open');
            overlay.classList.add('open');
            document.body.style.overflow = 'hidden';
        }
    }

    function closeSidebar() {
        const sidebar = document.querySelector('.session-sidebar');
        const overlay = document.querySelector('.sidebar-overlay');

        sidebar.classList.remove('open');
        overlay.classList.remove('open');
        document.body.style.overflow = '';
    }

    // 4. 监听窗口大小变化
    let lastWidth = window.innerWidth;
    window.addEventListener('resize', () => {
        const currentWidth = window.innerWidth;
        if (lastWidth <= 768 && currentWidth > 768) {
            // 从小屏切到大屏，关闭侧边栏
            closeSidebar();
        }
        lastWidth = currentWidth;
    });

    // 5. 触摸滑动关闭侧边栏
    let touchStartX = 0;
    document.addEventListener('touchstart', (e) => {
        touchStartX = e.touches[0].clientX;
    });

    document.addEventListener('touchend', (e) => {
        const touchEndX = e.changedTouches[0].clientX;
        const diff = touchStartX - touchEndX;

        // 从左向右滑动超过 50px，关闭侧边栏
        if (diff < -50 && document.querySelector('.session-sidebar.open')) {
            closeSidebar();
        }
    });

    // 6. 防止输入框被键盘遮挡
    const inputs = document.querySelectorAll('input, textarea');
    inputs.forEach(input => {
        input.addEventListener('focus', () => {
            setTimeout(() => {
                input.scrollIntoView({ behavior: 'smooth', block: 'center' });
            }, 300);
        });
    });

    // 7. 初始化
    document.addEventListener('DOMContentLoaded', () => {
        createMobileMenuBtn();
        createOverlay();
    });

})();
```

### 3.4 修改 API 基地址（自动适配）

修改 `frontend/js/common.js` 中的 API_BASE：

```javascript
// ==========================================
// API 地址配置 — 自动适配环境
// ==========================================

const API_BASE = (() => {
    // 1. 检测是否在 PWA/App 中运行
    const isStandalone = window.matchMedia('(display-mode: standalone)').matches 
                      || window.navigator.standalone === true;

    // 2. 检测是否在 Android WebView 中
    const isWebView = /wv|WebView/.test(navigator.userAgent) 
                     || (window.Android && window.Android.callNative);

    // 3. 检测是否在 localhost/127.0.0.1 访问
    const isLocalhost = location.hostname === 'localhost' 
                       || location.hostname === '127.0.0.1';

    // 4. 检测当前协议
    const isHttps = location.protocol === 'https:';

    // 配置优先级：
    // 1. 如果前端和后端在同一个域名（部署到服务器），用相对路径
    // 2. 如果是本地开发，用 localhost
    // 3. 如果是手机访问电脑内网，需要配置具体 IP

    if (isLocalhost) {
        return 'http://localhost:8000';
    }

    // 生产环境：前端和后端同域
    return '';  // 空字符串表示使用相对路径（同域）
})();

// WebSocket 地址
const WS_BASE = (() => {
    const isLocalhost = location.hostname === 'localhost' || location.hostname === '127.0.0.1';

    if (isLocalhost) {
        return 'ws://localhost:8000';
    }

    const protocol = location.protocol === 'https:' ? 'wss:' : 'ws:';
    return `${protocol}//${location.host}`;
})();

console.log('API_BASE:', API_BASE);
console.log('WS_BASE:', WS_BASE);
```

### 4.3 创建 Service Worker（离线缓存）

在 `frontend/` 创建 `sw.js`：

```javascript
const CACHE_NAME = 'medical-ai-v1';
const STATIC_ASSETS = [
    '/static/index.html',
    '/static/login.html',
    '/static/css/common.css',
    '/static/css/chat.css',
    '/static/js/common.js',
    '/static/js/chat.js'
];

// 安装时缓存静态资源
self.addEventListener('install', (event) => {
    event.waitUntil(
        caches.open(CACHE_NAME).then((cache) => {
            return cache.addAll(STATIC_ASSETS);
        })
    );
    self.skipWaiting();
});

// 激活时清理旧缓存
self.addEventListener('activate', (event) => {
    event.waitUntil(
        caches.keys().then((cacheNames) => {
            return Promise.all(
                cacheNames
                    .filter((name) => name !== CACHE_NAME)
                    .map((name) => caches.delete(name))
            );
        })
    );
    self.clients.claim();
});

// 拦截请求，优先从缓存读取
self.addEventListener('fetch', (event) => {
    const { request } = event;
    const url = new URL(request.url);

    // API 请求不走缓存
    if (url.pathname.startsWith('/api/')) {
        return;
    }

    // 静态资源优先缓存
    if (request.method === 'GET') {
        event.respondWith(
            caches.match(request).then((response) => {
                if (response) {
                    return response;
                }
                return fetch(request).then((fetchResponse) => {
                    if (fetchResponse.ok) {
                        const clone = fetchResponse.clone();
                        caches.open(CACHE_NAME).then((cache) => {
                            cache.put(request, clone);
                        });
                    }
                    return fetchResponse;
                });
            }).catch(() => {
                // 离线时返回缓存的 index.html
                if (request.mode === 'navigate') {
                    return caches.match('/static/index.html');
                }
            })
        );
    }
});
```

### 4.4 注册 Service Worker

在 `frontend/js/common.js` 中添加：

```javascript
// ==========================================
// PWA Service Worker 注册
// ==========================================

if ('serviceWorker' in navigator) {
    window.addEventListener('load', () => {
        navigator.serviceWorker.register('/static/sw.js')
            .then((registration) => {
                console.log('SW 注册成功:', registration.scope);
            })
            .catch((error) => {
                console.log('SW 注册失败:', error);
            });
    });
}

// 检测是否已安装 PWA
window.addEventListener('appinstalled', () => {
    console.log('PWA 已安装到主屏幕');
});
```

修改 `frontend/index.html`、 `login.html`、 `report.html`、 `map.html` 的 `<head>`：

## 7. Step 5 — 内网穿透（外网访问）

如果你想在任何地方都能用手机访问，需要把本地后端暴露到公网。

### 方案 A：cpolar（推荐，免费）

```bash
# 1. 注册 cpolar: https://dashboard.cpolar.com/
# 2. 下载安装（Windows/Mac/Linux 都有客户端）

# 3. 安装后获取 authtoken，在命令行执行：
cpolar authtoken xxxxxxx

# 4. 创建隧道，将本地 8000 端口映射到公网
cpolar http 8000

# 5. 会显示类似：https://abc123.cpolar.io
#    这就是你的公网地址
```

然后修改前端 `common.js`：

```javascript
const API_BASE = 'https://abc123.cpolar.io';  // 你的 cpolar 地址
```

### 方案 B：ngrok

```bash
# 1. 注册 ngrok: https://ngrok.com/
# 2. 下载并配置 authtoken
ngrok config add-authtoken xxxxx

# 3. 启动隧道
ngrok http 8000

# 4. 获取公网地址（如 https://abc.ngrok-free.app）
```

### 方案 C：部署到云服务器（最稳定）

如果你有阿里云/腾讯云服务器：

```bash
# 1. 把代码部署到服务器
# 2. 用 Nginx 反向代理
# 3. 配置域名 + HTTPS
# 4. 前端 API_BASE 改为域名地址

# Nginx 配置示例
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## 8. 常见问题

### Q1: 手机浏览器打开显示 "无法访问此网站"

**排查步骤**：

1. **确认后端监听 0.0.0.0**：
```bash
# 应该看到 0.0.0.0:8000，不是 127.0.0.1:8000
netstat -tlnp | grep 8000
```

2. **确认手机和电脑在同一 WiFi**：
```bash
# 电脑端查看 IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# 手机端查看 IP
# 设置 → WiFi → 点击当前网络 → 查看 IP
# 确保在同一网段（如都是 192.168.1.xxx）
```

3. **关闭电脑防火墙**：
```bash
# Windows: 控制面板 → Windows Defender → 关闭防火墙
# Mac: 系统设置 → 网络 → 防火墙 → 关闭
# Linux: sudo ufw disable
```

4. **测试连通性**：
```bash
# 在手机浏览器直接访问
http://电脑IP:8000/health

# 如果返回 JSON，说明网络通
```

### Q2: PWA "添加到主屏幕" 按钮不出现

**原因**：manifest.json 或 Service Worker 配置有误。  
**检查清单**：

```
□ manifest.json 路径正确（/static/manifest.json）
□ manifest.json 中 start_url 正确
□ sw.js 注册成功（浏览器控制台看日志）
□ HTTPS 环境（localhost 除外，iOS 必须 HTTPS）
□ icons 图片存在且路径正确
```

**调试方法**：
- Chrome 桌面版：F12 → Application → Manifest / Service Workers
- Chrome 手机版：chrome://inspect → 连接手机调试

### Q3: 页面样式错乱，布局不对

**解决**：
1. 确认 `common.css` 中的 `@media (max-width: 768px)` 已添加
2. 清除浏览器缓存后刷新
3. 在浏览器开发者工具中切换手机模式测试

### Q4: WebSocket 连接失败

**原因**：手机访问时 WebSocket 地址可能不对。  
**解决**：

```javascript
// common.js 中确认 WS_BASE 正确
const WS_BASE = (() => {
    const isLocalhost = location.hostname === 'localhost' || location.hostname === '127.0.0.1';

    if (isLocalhost) {
        return 'ws://localhost:8000';
    }

    // 生产环境自动适配
    const protocol = location.protocol === 'https:' ? 'wss:' : 'ws:';
    return `${protocol}//${location.host}`;
})();
```

### Q5: 输入框被键盘遮挡

**解决**：已在 `common.js` 中添加自动滚动代码：

```javascript
input.addEventListener('focus', () => {
    setTimeout(() => {
        input.scrollIntoView({ behavior: 'smooth', block: 'center' });
    }, 300);
});
```

如果仍有问题，在 `common.css` 中添加：

```css
.chat-input-bar {
    position: fixed;
    bottom: 0;
    padding-bottom: env(safe-area-inset-bottom);
}
```

### Q6: 图片上传/相机无法使用

**原因**：手机浏览器对 `<input type="file">` 有限制。  
**解决**：

```html
<!-- 在 input 中添加 accept 和 capture -->
<input type="file" 
       accept="image/*" 
       capture="environment"
       id="imageUpload">
```

- `capture="environment"`：调用后置相机
- `capture="user"`：调用前置相机
- 不加 capture：让用户选择相册或相机

### Q7: 语音输入（Whisper）无法使用

**原因**：浏览器录音需要 HTTPS（localhost 除外）。  
**解决**：
1. 使用内网穿透获得 HTTPS 地址
2. 或使用原生 App 壳调用系统录音 API

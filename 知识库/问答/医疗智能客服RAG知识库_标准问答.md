## 第一部分：挂号与预约

### Q001：如何预约挂号？
**答案：**
预约挂号有以下几种方式：
1. **线上预约**：
   - 医院官方微信公众号/小程序
   - 支付宝医疗健康平台
   - 第三方平台（如好大夫在线、微医等）
2. **电话预约**：拨打医院预约挂号热线
3. **现场预约**：凭有效身份证件到医院挂号窗口或自助挂号机办理
4. **诊间预约**：本次就诊时直接在诊室与医生预约下次复诊时间

温馨提示：预约成功后请按短信提示的时段到医院取号就诊，过时号源可能被释放。

---

### Q002：预约挂号需要准备哪些信息？
**答案：**
预约挂号通常需要准备以下信息：
1. **患者基本信息**：姓名、身份证号码、手机号码
2. **医保信息**：医保卡号（如需使用医保结算）
3. **就诊信息**：拟就诊科室、医生、日期
4. **症状描述**：简要描述主要不适症状（部分科室需要）

儿童挂号需提供监护人信息，无身份证的儿童可使用户口本或出生证明。

---

### Q003：预约成功后可以取消吗？如何取消？
**答案：**
可以取消预约。不同渠道取消规则如下：

1. **线上取消**：
   - 就诊日前一天23:59前可在原预约渠道取消
   - 部分医院支持就诊当天取消（可能需在规定时间前）
2. **电话取消**：拨打预约热线取消
3. **自助机取消**：持身份证到医院自助机取消

**注意**：
- 就诊当日一般不支持线上取消，需到医院窗口办理
- 一个自然月内累计取消预约超过3次，将被限制预约权限1-3个月
- 如已取号（含电子号），需到医院窗口办理退号

---

### Q004：没有身份证可以挂号吗？
**答案：**
没有身份证仍可挂号，但需提供有效身份证明：

1. **儿童**：户口本、出生医学证明、医保卡（带照片的社保卡）
2. **成人**：护照、港澳通行证、临时身份证、军官证、医保卡（部分医院）
3. **外籍人士**：护照、外国人永久居留身份证

急诊患者可先就诊，后补办身份登记。

---

### Q005：专家号太难挂怎么办？
**答案：**
建议您尝试以下策略：

1. **多平台尝试**：同时在医院官网、微信公众号、第三方平台搜索同一专家的号源
2. **关注退号**：预约不来的患者会释放号源，可尝试就诊前一天或当天早间刷新
3. **团队号源**：部分专家有团队出诊，可预约团队内其他医生
4. **改约普通号**：初诊患者建议先挂专科普通号，由医生初步诊断后转诊或预约专家
5. **复诊预约**：就诊结束时让医生直接预约下次复诊号源（诊间预约成功率更高）
6. **特需门诊**：专家特需门诊号源相对充足，费用较高但等候时间短

---

### Q006：医保用户如何挂号？
**答案：**
医保用户挂号需注意以下流程：

1. **首次就医**：需先到定点医院或通过医保APP关联医保账户
2. **挂号选择**：挂号时请选择"医保"通道，说明使用医保结算
3. **携带证件**：社保卡（电子医保凭证可替代实体卡）
4. **报销比例**：不同等级医院报销比例不同（通常一级医院>二级>三级）
5. **异地就医**：需提前在参保地医保局办理异地就医备案

---

### Q007：挂号后可以更换医生吗？
**答案：**
一般情况下，挂号后更换医生需要重新挂号：

1. **当日更换**：同一科室当日可找分诊台申请换号（视号源情况）
2. **换同科室其他医生**：需重新挂号或由分诊台协调
3. **换其他科室**：需重新挂号
4. **特殊情况**：如原医生临时停诊，医院会通知患者并优先安排同资质医生

建议就诊前确有需要时，先联系医院客服或分诊台咨询换号政策。

## 第二部分：就诊与取号

### Q008：预约成功后如何取号？
**答案：**
取号方式根据是否使用医保有所不同：

**自费患者取号方式：**
1. **自助机取号**：凭身份证或就诊码到医院自助挂号机打印挂号单
2. **窗口取号**：凭身份证到人工挂号窗口取号
3. **电子取号**：部分医院支持在线取号，就诊码直接作为凭证

**医保患者取号方式：**
1. 必须持**实体社保卡**或**医保电子凭证**到自助机或窗口读卡取号
2. 自助机会自动关联医保账户，扣除挂号费（报销部分）

**取号时间**：建议比预约时段提前15-30分钟取号

---

### Q009：取号时发现信息有误怎么办？
**答案：**
发现信息有误时请及时处理：

1. **姓名错误**：需携带有效证件到挂号窗口更正后重新挂号
2. **科室错误**：可到分诊台说明情况，协调换号或退号
3. **日期错误**：当日发现可直接重新预约正确日期的号
4. **医保信息错误**：需到医保办或挂号窗口更正医保信息

**注意**：就诊信息与本人不符可能影响医保结算和病历记录准确性，请务必核实。

---

### Q010：就诊流程是怎样的？
**答案：**
标准就诊流程如下：

1. **挂号/取号** → 凭预约信息或证件取号
2. **候诊签到** → 到达科室后用挂号单或就诊码在自助机/分诊台签到
3. **候诊等待** → 按叫号系统顺序等候（请保持安静）
4. **就诊问诊** → 向医生描述症状、病史，配合检查
5. **缴费检查/取药** → 凭医生开具的申请单缴费后进行检查或取药
6. **复诊/离院** → 检查结果出来后回诊室复诊，或遵医嘱离院

**温馨提示**：CT、MRI等大型检查需提前预约，当日可能无法完成。

---

### Q011：签到后还需要等很久，正常吗？
**答案：**
等候时间受多种因素影响，属于正常现象：

1. **预约时段是候诊区间**：预约号段（如9:00-9:30）内有多个患者，需按实际签到顺序排队
2. **前序患者耗时较长**：部分患者病情复杂，问诊时间较长
3. **医生临时处理紧急情况**：如有危急患者会优先处理
4. **检查结果回示**：部分患者需等检查结果后再回诊

**建议**：
- 签到后可在候诊区等候，注意叫号屏幕
- 如等候超过1小时，可联系分诊台咨询
- 大型检查完成后需回诊室复诊，通常需重新排队

---

### Q012：急诊和普通门诊有什么区别？
**答案：**
急诊与普通门诊在以下方面有显著区别：

| 对比项 | 急诊 | 普通门诊 |
|--------|------|----------|
| 收治范围 | 急性疾病、意外伤害、病情危重 | 常见病、慢性病、复诊 |
| 开放时间 | 24小时 | 固定工作时间 |
| 挂号方式 | 现场挂号，不接受预约 | 预约+现场挂号 |
| 费用 | 通常较高（含急诊服务费） | 按标准门诊收费 |
| 排队方式 | 护士分诊后按病情分级 | 预约时段叫号 |
| 检查速度 | 优先处理，快速出结果 | 按常规流程 |

**请根据病情选择**：危重症（胸痛、昏迷、大出血等）直接去急诊；一般不适建议预约门诊。

---

### Q013：医生让我做检查，但当天做不了怎么办？
**答案：**
检查当天无法完成时，通常有以下处理方式：

1. **改约检查日期**：
   - 缴费后凭检查申请单到相应检查科室预约具体时间
   - 部分检查（CT、MRI、胃肠镜等）需要提前预约
2. **先回家等通知**：
   - 大型检查（如PET-CT）预约周期较长，回家等候，护士会电话通知
3. **急查绿色通道**：
   - 如病情紧急，医生会标注"急查"，检查科室优先安排
4. **异地检查**：
   - 部分医院可开具检查申请单到合作机构完成

**注意**：完成检查后，请按约定时间取报告并回诊室复诊。

## 第三部分：检验检查

### Q014：检查报告多久可以取？
**答案：**
不同检查项目出报告时间不同：

| 检查类型 | 通常出报告时间 |
|----------|--------------|
| 血常规 | 30分钟-2小时 |
| 生化检查（肝肾功能等） | 当日或次日 |
| 免疫检查（肝炎、肿瘤标志物） | 1-3个工作日 |
| 病理报告 | 3-7个工作日 |
| CT平扫 | 2-4小时 |
| CT增强/MRI | 24-48小时 |
| B超（普通） | 当场出报告 |
| 心电图 | 当场出报告 |
| 胃镜/肠镜 | 当场出报告，活检3-7天 |

**取报告方式**：自助机打印、微信查询、护士站领取

---

### Q015：抽血检查需要空腹吗？
**答案：**
是否空腹取决于检查项目：

**需要空腹的检查（禁食8-12小时）：**
- 血糖、血脂系列
- 肝功能、肾功能
- 腹部B超（肝胆脾胰）
- 胃镜、肠镜

**不需要空腹的检查：**
- 血常规、凝血功能
- 甲状腺功能
- 心肌酶谱
- 尿常规
- 大部分激素检查

**温馨提示**：
- 空腹期间可少量饮用白开水
- 高血压患者照常服用降压药（用少量水送服）
- 糖尿病患者如有降糖药请咨询医生
- 不确定时请咨询开单医生或检验科

---

### Q016：CT检查有辐射吗？安全吗？
**答案：**
CT检查确实存在辐射，但一般来说是安全的：

**辐射剂量参考：**
| 检查类型 | 有效剂量（mSv） | 相当于自然本底辐射时间 |
|----------|--------------|-------------------|
| 胸部X线 | 0.1 | 约10天 |
| 头部CT | 2 | 约8个月 |
| 胸部CT | 7 | 约2年 |
| 腹部CT | 10 | 约3年 |

**安全性说明：**
- 常规CT检查的辐射剂量在安全范围内
- 医生会根据病情权衡利弊，开具必要的检查
- 孕妇应提前告知医生，权衡利弊后决定
- 短期多次CT检查需由医生严格评估

**注意事项**：CT增强检查需注射造影剂，有极小概率过敏反应，检查后请多饮水促进造影剂排泄。

---

### Q017：做B超需要憋尿吗？
**答案：**
不同B超部位对憋尿要求不同：

**需要憋尿的B超：**
- 经腹壁妇科B超（子宫、附件）：需膀胱充盈作为透声窗
- 泌尿系统B超（膀胱、输尿管下段）：需充盈膀胱

**不需要憋尿的B超：**
- 腹部B超（肝胆脾胰）：通常需空腹
- 甲状腺B超、乳腺B超：无需特殊准备
- 心脏B超、血管B超：无需特殊准备
- 经阴道B超：无需憋尿（检查前需排空膀胱）
- 产科NT检查：通常不需憋尿

**憋尿程度**：有尿意但不过度憋胀为宜（约喝水后30-60分钟）

---

### Q018：病理报告为什么需要那么久？
**答案：**
病理报告时间较长是由病理诊断的特殊性决定的：

**标准病理流程：**
1. 标本固定（4-24小时）：用福尔马林固定组织
2. 脱水透明（8-12小时）：逐步用酒精、二甲苯处理
3. 浸蜡包埋（4-6小时）：用石蜡包裹组织
4. 切片制片（1-2天）：切成3-5微米的薄片，染色
5. 病理阅片（1-3天）：病理科医生在显微镜下观察诊断
6. 疑难病例会诊：复杂病例需多位医生讨论或免疫组化染色

**加急情况**：
- 术中冰冻切片：30-60分钟（仅用于术中快速诊断）
- 普通病理：3-7个工作日为正常周期

病理报告是疾病诊断的"金标准"，需要足够时间保证准确性，请耐心等待。

## 第四部分：取药与用药

### Q019：取药流程是怎样的？
**答案：**
取药标准流程：

1. **缴费**：凭医生开具的处方或申请单到收费窗口/自助机缴费
2. **获取取药凭证**：缴费后系统自动分配取药窗口，打印凭条或收到短信提示
3. **前往药房**：根据凭条显示的窗口号，到相应药房取药
4. **核对药品**：取药时药师会核对姓名，请确认无误后签字
5. **用药指导**：取药时请咨询药师用药方法和注意事项

**取药地点**：通常在医院一楼药房，部分医院有中药房、西药房分设

---

### Q020：药品用法用量怎么理解？
**答案：**
处方上的用法用量说明：

**常见用法术语：**
| 术语 | 含义 |
|------|------|
| 每日3次（tid/q8h） | 每8小时服用一次，24小时内3次 |
| 每日2次（bid/q12h） | 每12小时服用一次，24小时内2次 |
| 每日1次（qd） | 每日固定时间服用1次 |
| 睡前服（qn） | 晚上睡前服用 |
| 饭前服（ac） | 餐前30分钟服用 |
| 饭后服（pc） | 餐后15-30分钟服用 |
| 必要时服（prn） | 症状出现时服用，遵医嘱限制频次 |

**服用方式**：
- **口服**：片剂/胶囊直接温水送服，不要干吞
- **外用**：皮肤用药遵医嘱薄涂或湿敷，眼药水注意无菌操作
- **雾化**：按说明书配制，正确使用雾化器

**重要提醒**：请严格按照医嘱服药，不要自行增减剂量或停药。

---

### Q021：药品需要冷藏吗？
**答案：**
是否冷藏取决于药品性质：

**需要冷藏（2-8℃）的常见药品：**
- 胰岛素
- 部分生物制剂（如重组人粒细胞刺激因子）
- 某些活菌制剂（如双歧杆菌三联活菌散）
- 部分眼药水（开封后需冷藏）
- 某些靶向药物

**常温保存（10-30℃）的常见药品：**
- 大部分口服片剂、胶囊
- 普通外用药膏
- 常规口服液

**保存注意事项：**
- 冷藏药品避免结冰（结冰后药品失效）
- 外出携带请使用冰袋保温
- 夏季高温请注意药品不要放在车内
- 每次使用前检查药品外观，如有变色、沉淀请停用并咨询药师

---

### Q022：忘记服药怎么办？
**答案：**
不同药品处理原则不同：

**一般原则：**
- **发现漏服时距下次服药时间>6小时**：可立即补服一次剂量
- **发现漏服时距下次服药时间<6小时**：跳过本次，不补服，下次按原时间服药
- **切勿加倍服用**：一次服用两倍剂量可能引发不良反应

**特殊药品漏服处理：**
| 药品类型 | 漏服处理建议 |
|----------|------------|
| 降压药 | 根据血压情况，咨询医生 |
| 降糖药 | 特别是餐前服药，漏服后需根据血糖调整 |
| 抗生素 | 漏服立即补服，确保血药浓度 |
| 激素类 | 不可随意补服，需咨询医生 |
| 抗癫痫药 | 需咨询医生，不可自行补服 |

**预防措施**：建议设置手机闹钟或使用分药盒提醒。

---

### Q023：中药怎么煎煮？
**答案：**
中药煎煮方法：

**常规煎煮步骤：**
1. **浸泡**：将中药倒入砂锅/陶瓷锅，加冷水没过药面3-5cm，浸泡30分钟
2. **第一煎**：大火煮沸后转小火，煎煮20-30分钟，倒出药液（约200ml）
3. **第二煎**：再加冷水，同上法煎煮15-20分钟，倒出药液
4. **合并**：将两次药液混合，分早晚两次服用

**特殊煎煮方法：**
| 类型 | 煎法 |
|------|------|
| 先煎 | 先煎煮20-30分钟，再加入其他药材 |
| 后下 | 煎煮结束前5-10分钟加入 |
| 包煎 | 用纱布袋包好后煎煮 |
| 烊化 | 用热水或药液融化后服用 |
| 冲服 | 直接用药液或温水冲服 |

**器具选择**：推荐砂锅、陶瓷锅，禁用铁锅、铝锅。
**注意事项**：请遵医嘱，部分药材有特殊煎法要求。

---

### Q024：药品有副作用吗？
**答案：**
所有药物都有可能出现副作用，这是正常现象：

**常见副作用类型：**
| 系统 | 常见表现 | 处理建议 |
|------|---------|----------|
| 胃肠道 | 恶心、呕吐、腹泻 | 饭后服用，或告知医生换药 |
| 皮肤 | 皮疹、瘙痒 | 停药并就医 |
| 神经系统 | 头晕、嗜睡 | 避免驾驶和高空作业 |
| 肝脏 | 肝功能异常 | 定期监测肝功能 |
| 血液 | 白细胞减少、血小板减少 | 定期复查血常规 |

**处理原则：**
1. 大多数轻微副作用在服药数天后可自行缓解
2. 如症状持续或加重，请及时就医
3. 出现严重过敏反应（呼吸困难、全身皮疹等）请立即就医
4. 记住药名，复诊时告知医生不良反应情况

**重要提示**：请勿因担心副作用而自行停药，应与医生沟通后决定。

## 第六部分：费用与医保

### Q028：门诊费用如何报销？
**答案：**
门诊费用报销根据参保类型有所不同：

**城镇职工医保（职工医保）：**
- 挂号费：可直接报销（需持社保卡挂号）
- 药品费：按药品目录报销，甲类全额计入、乙类部分计入、丙类自费
- 检查费：按项目报销，部分大型检查需先行自付一定比例
- 起付线：各地不同（通常300-1500元/年），超过部分按比例报销
- 封顶线：通常2-4万元/年

**城乡居民医保（居民医保/新农合）：**
- 报销比例低于职工医保
- 通常需要在定点医院就诊才能报销
- 部分地区已实行门诊统筹

**报销方式**：
- **实时结算**：持社保卡就医，系统自动结算（仅限医保目录内项目）
- **事后报销**：异地就医或自费后，凭票据到医保局报销

---

### Q029：医保卡没钱了还能报销吗？
**答案：**
可以报销。医保卡账户余额与报销资格是不同概念：

**区分说明：**
- **医保个人账户余额**：用于支付医保目录内需要自付的部分（可理解为"自己的钱"）
- **医保统筹基金**：报销时由医保基金支付的部分（符合条件即可报销）

**举例说明**：
- 某次门诊费用200元，医保目录内150元，报销比例60%
- 统筹报销：150×60%=90元（由医保基金支付）
- 个人自付：150×40%=60元（可用个人账户余额支付）
- 丙类自费：50元（完全自费）

**结论**：只要本年度未超封顶线、就诊医院为定点医院、费用属于医保目录内，无论个人账户是否有钱，都可以按规定报销。

### Q030：住院押金怎么算？可以退吗？
**答案：**
住院押金相关说明：

**押金计算方式：**
- 通常为预计住院费用的50%-100%
- 不同科室押金标准不同（手术科室押金较高）
- 医保患者押金可能低于自费患者

**押金用途：**
- 支付住院期间的自费项目
- 支付超出医保报销范围的部分

**押金退还：**
- 出院结算时多退少补
- 押金收据是退款的唯一凭证，请妥善保管
- 出院后7-14个工作日内可办理退款（以各医院规定为准）

**注意事项**：
- 押金不足会影响后续治疗，请及时续缴
- 出院前请确认所有费用已计入，避免遗漏
- 押金收据丢失需到住院处办理挂失手续

### Q031：医保账户家庭共济怎么办理？
**答案：**
医保个人账户家庭共济是指将个人账户余额授权给家人使用：

**办理条件（各地略有差异）：**
- 授权人需参加职工医保，且个人账户有余额
- 被授权人需参加基本医保（职工或居民医保均可）
- 双方需在同一统筹区参保

**办理渠道：**
1. **线上办理**：国家医保服务平台APP、当地医保局官方公众号/小程序
2. **线下办理**：医保中心服务窗口

**绑定流程（以国家医保服务平台APP为例）：**
1. 下载并登录国家医保服务平台APP
2. 进入"地方专区"选择参保地
3. 点击"个人账户共济"→"共济账户授权"
4. 填写被授权人信息，确认绑定

**使用方式**：被授权人在就医结算时，系统自动扣除授权人个人账户余额

## 第八部分：常见疾病问答

### Q034：发烧了怎么办？
**答案：**
发热（发烧）的处理原则：

**什么情况下需及时就医：**
- 体温超过39℃
- 持续发热超过3天
- 伴有抽搐、皮疹、呼吸困难、意识模糊等症状
- 儿童、孕妇、老年人等特殊人群发热
- 有基础疾病的患者（心肺疾病、糖尿病等）

**家庭处理建议：**
1. **监测体温**：每4-6小时测一次体温
2. **物理降温**：体温38.5℃以上可温水擦浴（颈部、腋下、腹股沟）
3. **药物退热**：体温38.5℃以上可服用对乙酰氨基酚或布洛芬（注意用药间隔）
4. **补充水分**：多喝温水，防止脱水
5. **充足休息**：保持室内通风，穿着宽松衣物

**注意事项**：
- 退热药需间隔4-6小时，不可短期内重复大量服用
- 儿童禁止使用阿司匹林退热
- 不建议自行服用抗生素，发热原因需医生诊断

### Q037：高血压患者饮食需要注意什么？
**答案：**
高血压患者的饮食管理建议：

**推荐遵循"DASH饮食"原则：**
1. **减少钠盐摄入**：
   - 每日钠盐摄入<5克（约一啤酒瓶盖）
   - 少食用腌制食品、加工肉类
   - 使用低钠盐（肾功能正常者）
2. **增加钾摄入**：
   - 多吃新鲜蔬菜水果（如香蕉、菠菜、橙子）
   - 肾功能正常者可适当补钾
3. **控制脂肪摄入**：
   - 少吃肥肉、动物内脏
   - 烹调油每日<25克
   - 选择橄榄油、茶籽油等健康油脂
4. **适量蛋白质**：
   - 每天1g/kg体重
   - 优先选择鱼、禽、瘦肉、豆制品
5. **限制酒精**：男性<25g/天，女性<15g/天
6. **控制糖分**：少吃甜食、含糖饮料

**推荐食物**：全谷物、蔬菜、水果、低脂奶制品、坚果、鱼类

### Q038：糖尿病患者血糖控制目标是多少？
**答案：**
糖尿病患者血糖控制目标（一般人群）：

**血糖控制目标：**
| 指标 | 目标值 |
|------|--------|
| 空腹血糖 | 4.4-7.0 mmol/L |
| 餐后2小时血糖 | <10.0 mmol/L |
| 糖化血红蛋白（HbA1c） | <7.0% |
| 睡前血糖 | 5.6-7.8 mmol/L |

**特殊人群目标调整：**
| 人群 | HbA1c目标 |
|------|-----------|
| 病程短、无并发症、预期寿命长 | <6.5% |
| 老年患者、严重低血糖史 | <8.0% |
| 合并严重并发症 | <8.5% |

**监测频率**：
- 口服药治疗：每周测1-2天空腹+餐后血糖
- 胰岛素治疗：每日测血糖（根据方案调整）
- 每3个月测一次糖化血红蛋白

**重要提示**：血糖目标需个体化制定，请遵医嘱执行。

### Q039：感冒了需要吃抗生素吗？
**答案：**
普通感冒不需要吃抗生素。

**感冒类型与治疗原则：**
| 类型 | 病因 | 是否需要抗生素 |
|------|------|--------------|
| 普通感冒（病毒性） | 鼻病毒、冠状病毒等 | ❌ 不需要 |
| 流感 | 流感病毒 | ❌ 不需要（可用抗病毒药如奥司他韦） |
| 细菌性上呼吸道感染 | A组链球菌等 | ✅ 需要（需医生处方） |
| 细菌性肺炎 | 细菌 | ✅ 需要（需医生处方） |

**感冒自我处理建议：**
1. 多休息、多饮水
2. 对症处理：发热用退热药，鼻塞用减充血剂
3. 如症状持续超过10天、反复发热、出现黄绿脓涕，需就医评估是否合并细菌感染

**抗生素使用原则**：
- 抗生素对病毒无效，滥用会导致耐药
- 是否有细菌感染需医生判断，不建议自行购买抗生素
- 如医生开具抗生素，请完成整个疗程

### Q040：体检发现肺部结节怎么办？
**答案：**
肺结节的处理原则：

**什么是肺结节？**
肺结节是指CT上发现的直径≤3cm的肺部阴影，大于3cm称为肿块。

**结节大小与随访策略（以实性结节为例）：**
| 结节大小 | 处理建议 |
|----------|---------|
| <4mm | 低危人群年度随访，高危人群6-12个月随访 |
| 4-6mm | 6-12个月复查，稳定后年度随访 |
| 6-8mm | 3-6个月复查，评估生长速度 |
| >8mm | 进一步评估（PET-CT、穿刺活检等） |

**高危因素**：
- 长期吸烟史
- 年龄>55岁
- 肺癌家族史
- 职业暴露史（石棉、氡气等）
- 有慢性肺病史

**重要提示**：
- 大多数肺结节是良性的，不必过度恐慌
- 随访过程中结节增大或形态变化需高度警惕
- 请遵医嘱定期复查，不要自行中断随访
- 戒烟是预防肺结节恶变的有效措施

| 尿频、尿急、尿痛 | 泌尿外科/肾内科 |
| 皮疹、瘙痒 | 皮肤科 |
| 失眠、情绪低落 | 心理科/神经内科 |
| 儿童各类症状 | 儿科 |

**建议**：如无法判断，首选全科医学科或普通内科就诊，由医生分诊。

### Q042：智能预诊系统能帮我做什么？
**答案：**
智能预诊系统主要提供以下服务：

**主要功能：**
1. **症状采集与分析**
   - 收集患者主诉症状、发病时间、伴随症状
   - 分析症状特征，评估可能病因
   
2. **科室推荐**
   - 根据症状推荐合适的就诊科室
   - 标注需要急诊的情况（如胸痛、腹痛剧烈等）

3. **病情分级**
   - 初步评估紧急程度
   - 标注"建议立即就医"或"可择期就诊"

4. **预填病史**
   - 将预诊信息自动同步到门诊病历
   - 节省医生问诊时间

5. **健康科普**
   - 根据症状推送相关健康知识
   - 解答常见问题

**局限性**：
- 智能预诊仅供参考，不能替代医生诊断
- 复杂病情仍需医生面诊评估
- 危急重症请直接到急诊

## 附录：常用医疗术语解释

### A001：什么是挂号费/诊查费？
**答案：** 挂号费（诊查费）是患者到门诊就诊时支付的服务费用，用于补偿医生提供诊疗服务的劳动。费用根据医生职称（主治医师、副主任医师、主任医师）和门诊类型（普通门诊、专家门诊、特需门诊）有所不同。部分省市已实行医事服务费制度，替代挂号费。

### A002：什么是医保目录（药品目录/诊疗项目目录）？
**答案：** 医保目录是国家或地方医保部门制定的报销范围清单，包括：
- **药品目录**：哪些药品可以报销（分甲类、乙类、丙类）
- **诊疗项目目录**：哪些检查治疗项目可以报销
- **服务设施目录**：床位费等服务设施的报销标准
不在医保目录内的项目，通常需要自费。

### A003：什么是起付线和封顶线？
**答案：** 
- **起付线**：医保报销前患者需先自付的费用额度，超过起付线的部分才进入报销范围。各地标准不同，通常门诊300-1500元/年，住院800-2000元/次。
- **封顶线**：医保报销的最高限额，超过封顶线的费用需自费。各地政策不同，通常门诊2-4万/年，住院20-50万/年。

### A004：什么是DRG/DIP付费？
**答案：** DRG（按疾病诊断相关分组）和DIP（按病种分值付费）是医保支付方式改革的两种模式：
- **DRG**：根据患者的疾病诊断、治疗方式、年龄等因素，将病例分入若干组，每组设定支付标准
- **DIP**：根据病种和治疗方式的分值计算支付金额
这两种方式不同于传统的"按项目付费"，有助于控制医疗费用不合理增长，推动医院提高诊疗效率。
